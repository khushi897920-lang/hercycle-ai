# GitHub Issue Labels Reference

This document catalogs and defines all labels used in the HerCycle AI issue tracker. Use these labels to filter and find issues relevant to your skill level and area of interest.

---

## Core Labels

### `ECSOC`
* **Purpose:** Identifies issues that are officially participating in the ECSOC open-source event.
* **Color:** `#C2185B` (Crimson)
* **Audience:** ECSOC contributors.

---

## Difficulty Levels

### `good first issue`
* **Purpose:** Simple, low-risk changes suitable for first-time contributors to the project.
* **Color:** `#7057ff` (Purple)

### `beginner`
* **Purpose:** Requires basic understanding of CSS, markdown, or JavaScript.
* **Color:** `#0072b2` (Blue)

### `intermediate`
* **Purpose:** Requires familiarity with Next.js page state, React hooks, API endpoints, or database structures.
* **Color:** `#e69f00` (Amber)

### `advanced`
* **Purpose:** Complex tasks involving AI pipelines, Clerk webhook verifications, database RLS, and performance indices.
* **Color:** `#d55e00` (Orange-Red)

---

## Categorization Labels

### `bug`
* **Purpose:** Something isn't working as expected. Fixes a crash, incorrect calculation, or logical flaw.
* **Color:** `#d73a4a` (Red)

### `enhancement`
* **Purpose:** Proposes a new feature, dashboard panel, or algorithmic addition.
* **Color:** `#a2eeef` (Light Blue)

### `documentation`
* **Purpose:** Improves, corrects, or translates Markdown docs, setup guides, and inline code comments.
* **Color:** `#0075ca` (Dark Blue)

### `ui` / `ux`
* **Purpose:** Modifies CSS styling, glassmorphism layouts, responsiveness, transitions, or typography.
* **Color:** `#fef2c0` (Yellow)

### `performance`
* **Purpose:** Improves route response times, queries, page generation, or build speeds.
* **Color:** `#ecc5f2` (Pink)

### `security`
* **Purpose:** Fixes session checks, validates inputs, mitigates leakages, or hardens APIs.
* **Color:** `#0e8a16` (Green)

### `testing`
* **Purpose:** Adds unit tests, mock request payloads, or expands testing scripts.
* **Color:** `#fbca04` (Orange-Yellow)

### `help wanted`
* **Purpose:** The maintainers are looking for additional help or feedback on a complex design decision.
* **Color:** `#008672` (Teal)
